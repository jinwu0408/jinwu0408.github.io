What is Supreme Bot?

Supreme is an American skateboarding shop and clothing brand established in New York City in April 1994. They are known for their fast sell-out speed. Because some of their items sells really fast due to the demand, many people are willing to pay ten times the price in the second market. As a collector of Supreme, I don't want to pay the ridulous resell price. So,I made this Supreme Bot helps to add items in cart and checkout automatically in a timely fashion.

Setup Profile

Use a separate file to store all the user infomation that will be filled. Including:

  • Item
    • Keyword
    • Size
    • Color
  • Personal Information
    • Name
    • Address
    • Phone Number
    • Card Info.

Find The Correct Item

We are using a pytho library called Webdriver, which is used to drive Google Chrome. First, we need to load the Supreme page in Chrome. I noticed that supreme URL is labeled by the category afterfix, such as https://supremenewyork.com/shop/t-shirt. Second, we need to load item infomation and find the element. Once we found the correct item, then we get the "href" attribute from the element. Then we redirect to item page.

Add To Cart

Now, we are in the item page, then we need to add item into cart. First we have to select the correct size. Then, we need to click on two button. First, "Add to Cart" button. Then, "Checkour Now" button. once it is clicked, we are redirect to the checkout page

Check Out

To checkout the item, we need to use the methods from Webdriver, send_key(), Select(), to fill all the infomation automatically. Then, we just use .click() method to click the checkout button. then we are done. The whold process takes about 5 seconds.
