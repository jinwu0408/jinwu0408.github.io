AMC Messenger Bot

What is AMC Messenger Bot?

AMC messenger bot is a facebook messenger bot that auto load lastest movie showtimes from AMC website by webscraping. So far, it only utilizes AMC. However, it has the potential to have more functions, such as local events search, concert tickets booking, even shopping for daily supplies.

Echo Bot

First we need to make sure that we can communicate through facebook's API by making an echo bot using Facebook developer tool and python library-Pymessenger. We used Flask to make web services and glitch for real time hosing.

The echo bot will echo back what ever the user's messages are. Overall structure is similiar to the picture below:

Whenever the user sent a message, the flask server would capture it and sent it to our bot. Our bot, then copy the message and send it back to the user by making a post request to facebook messenger.

Webscraping

After we are able to receive and sent messages, we need to prepare respond. In other word, we need to get the showtime of all the current movies.

We do this by webscraping AMC's website. First we need to land on the showtime page. AMC's showtime button turn grey, if the time is already passed. Notice, in the picture below, only 9:20pm is white, and it is because this screenshot was taken at 9:00pm.

I don't want send showtimes that are already pass. So the first thing is looping through all the showtimes and filter out passed times. Then, we read the movie titles, and all their respected showtimes. Last, we get the "href", which is the tickets buying pages.

Once we have all the Informations, we prepare the button message that users can just click on it and it redirct them to the tickets page.
